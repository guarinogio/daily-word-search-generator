# Contributing

## Setup

```bash
npm install
cp .env.example .env
```

Fill in your Gemini API key.

## Development

Run CLI:

```bash
npm run dev -- --help
```

Run checks:

```bash
npm run check
npm run lint
npm run format
```

## Commit style

Use simple semantic prefixes:

- feat:
- fix:
- chore:
- docs:

## Pull requests

- Ensure lint and type check pass
- Keep changes focused
- Update README if needed
